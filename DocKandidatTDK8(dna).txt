Smartphone as a breakthrough in information technology is much sought after by various parties. One of them in the field of tourism. The existence of a yet unknown location, tourists will be the difficulty in finding such tourist sites. Therefore, made an application-based LBS to help tourists get tourist and travel information.
This will be built on the research of an application on LBS the Android operating system which allows the user to know tourist sites. Technology LBS used for know the location of users with position location will be targeted. This application is able to calculate the distance and sort the list based on distance to the nearest tourist location. The application can also display routes and navigation services using Google Map API. The application has features features call, the share feature and the feature opens up a web address. The application also implemented Google Anaytic to monitor the use of the application.
Based on the test results, all of the functionality mentioned above can be met by the application have been made. The result of the implementation of the Google Analytic has also shows data about the usage of the application. Additionally application has also been successfully tested on a few different Android devices.
